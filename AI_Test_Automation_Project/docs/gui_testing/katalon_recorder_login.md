# Katalon Recorder Login Test

This file documents a sample GUI test recorded with Katalon Recorder for Sauce Demo.

## Recorded steps

1. Open URL: `https://www.saucedemo.com/`
2. Type `standard_user` into `id=user-name`
3. Type `secret_sauce` into `id=password`
4. Click `id=login-button`
5. Assert that the current URL contains `inventory.html`

## Purpose

This confirms that at least one GUI test case has been implemented using Katalon Recorder.
The same flow is also supported by the repository's Selenium WebDriver runner.
